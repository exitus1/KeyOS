# `simulator-cli`

## Commands
- 'd': device screenshot
- 's': simple screenshot
- 'rd': record device
- 'rs': record screen
- 'c': cut recording
- '1x': set scale to 1x
- '1.5x': set scale to 1.5x
- '2x': set scale to 2x
- 'td': theme dark
- 'tl': theme light
