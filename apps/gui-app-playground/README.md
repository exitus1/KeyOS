# Developer Playground

### Loader Playground
- Test three different loader types:
  - Continuous Arc (TimerCircularProgress)
  - Timer Loader (TimerLoader)
- Configurable parameters:
  - Size (100-400px)
  - Thickness (2-30% for arc loaders)
  - Tick interval (50-500ms for discrete loaders)
- Dark/Light mode toggle

### Haptics Playground
- Test all available haptic patterns
- Interactive pattern selection
- Immediate haptic feedback

### LED Control
- Control all 4 device LEDs
- RGB color picker with sliders
- Real-time color preview
- Individual LED selection
